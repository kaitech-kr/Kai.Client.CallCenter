# 기본 수칙

## 코드 작업 규칙

1. **절대로 허락없이 임의로 코드를 수정하거나, 삭제하거나, 새로 만들지 않는다**

2. **작업은 한번에 한개의 함수만 한다**

3. **함수가 길때엔 나누어서 확인 받으면서 작업한다**

4. **nul파일을 만들지 않는다**

5. **부득이하게 만들었을땐 delete_nul.py로 즉시 삭제한다**

6. **주석 처리된 함수는 개선점을 찾고 리팩토링할 계획이므로, 항상 개선점을 찾아본 후에 리팩토링한다**

7. **함수 선언은 모든 매개변수를 포함하여 수평(한 줄)으로 작성한다**
   - 예시: `private async Task<StdResult_Status> SearchAndSelectCustomerAsync(IntPtr hWnd고객명, IntPtr hWnd동명, string custName, string chargeName, string fieldName, CancelTokenControl ctrl)`
   - 매개변수가 많아도 수평으로 유지
